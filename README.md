## CodeFly

### Installation

    go get github.com/Arror/CodeFly

    cd .../GOPATH/src/github.com/Arror/CodeFly

    make build

### Usage

    CodeFly <Generate commond> -l <Language> -i <Input thrift file path.> -o <Output files path.>

### Requirements

Go 1.8
