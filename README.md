## CodeFly

### Installation

    go get github.com/Arror/CodeFly

    cd .../GOPATH/src/github.com/Arror/CodeFly

    make build

### Usage

    CodeFly json -l swift -i ./example.thrift -o ./

### Requirements

Go 1.8
